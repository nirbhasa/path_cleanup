
Path Cleanup Module

Description
------------
Drupal URL paths with slashes and dots can cause problems after the node is saved. This module removes slashes and dots from node paths before paths are saved.


Installation
-------------
1. Place the entire module directory into your Drupal sites/all/modules/
   directory.

2. Enable the module by navigating to:
     administer > modules
